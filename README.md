# Arpeggiator

An app that uses Tone.js to create an arpeggiator. An arpeggiator automatically steps through a sequence of notes based on an input chord.

**Programming Language**: JavaScript

**Library**: [Tone.js](https://tonejs.github.io/)

**Markup/Styling**: HTML5/SCSS

---

# Setup

- Download the files

- Install live-server:

```
npm install -g live-server
```
- Navigate to project folder and run:

```
live-server
```
